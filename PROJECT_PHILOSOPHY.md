# Project Philosophy

Maintainers have helped users for years.

Maintainer Flowboard exists to help maintainers keep their workspace clear.

This tool does not replace maintainers.
It does not make decisions for maintainers.
It does not close issues, merge pull requests, or post replies automatically.

Its job is smaller and more respectful:

> Organize information before human judgment.

The goal is:

**Less triage fatigue. More maintainer clarity.**
