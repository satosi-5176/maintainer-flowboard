Public GitHub import

Import public issues and PRs

This helper fetches public GitHub issues and pull requests, stores them in the same local board data used by Maintainer Flowboard, then lets you return to the board. It does not use a token, does not write to GitHub, and only works for public repositories.

Back to board
Enter a public repository such as owner/repo, then fetch.

GitHub unauthenticated API rate limits can apply. If fetching fails, wait and try again or use pasted JSON in the main board.
