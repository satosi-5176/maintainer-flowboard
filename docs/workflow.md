# Workflow

1. Export or collect issue/PR data.
2. Paste JSON into Maintainer Flowboard.
3. Review automatic classification.
4. Move cards manually where needed.
5. Add maintainer notes.
6. Generate weekly Markdown summary.
7. Use the summary as a private planning aid.

Do not treat classification as truth. Treat it as a starting point.

## Principle

Maintainer Flowboard does not make decisions. It organizes information before maintainer judgment.
